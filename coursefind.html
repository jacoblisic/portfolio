    In my Spring 2017 Human-Computer Interaction course, my group and I were tasked with creating a prototype of an idea of our choosing. We came up with a website that would pull courses from multiple online course websites into one. For example, this website would take courses from Codecademy, Udemy, or Lynda.com and display them all on one site for a user to view.

    What I contributed:

    • Helped define the project description of coursefind.
    • Distributed tasks to group members in order to complete the project on time.
    • Worked as a team to conduct user research.
    • Designed the website prototypes.
    • Created the final presentation to present to the class.
