    In my Spring 2017 Advanced Responsive Web Design course, we were tasked with creating a new website for the Kent Historical Society and Museum. Our class was divided into three teams, each with a team leader. These three teams were the UX team, the Design team, and the CMS team. My role was the team lead for the CMS team.

    What I contributed:

    • Researched old KHS website, finding which pages were visited the most and which were visited the least.
    • Led CMS team updates and meetings.
    • Assigned team members tasks to be completed on deadline.
    • Presented to the client regarding the CMS team's progress.
    • Created the HTML and CSS for the header navigation.
    • Constructed the global fields for the CMS.
    • Worked on integrating the global fields in the header, footer, and on multiple pages.
    • Edited the code for the globals and singles pages so that content was outputted on the website.
    • Transferred the CSS classes to the CMS layouts so the design was outputted correctly on the website.
    • Helped the UX team with troubleshooting CMS fields that were not working.
